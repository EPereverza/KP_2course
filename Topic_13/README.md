### Тема 13
### Задание
```
Используйте сервисы

Yandex Forms
Albato
GigaChat API
Yandex Почта
Google Sheets
Организуйте автоматизированную проверку теста, созданного в Yandex Forms.

Бизнес процесс:

Отправка ответа из Yandex Forms в связку Albato.
Извлечение ФИО и email ученика из JSON.
Извлечение ответов ученика из JSON.
Формирование и отправка сообщения в GigaChat через GigaChat API (нужно получить от Sber - Authorization Key).
Получение ответа от GigaChat.
Запись ответов ученика и комментария "преподавателя" (нейросети) в таблицу Google Sheets.
Отправка ответа нейросети ученику на почту.
Создайте в Albato настроенную связку из перечисленных в указанном порядке сервисов и инструментов работы с данными (в частности с JSON). Самый первый шаг в связке - это получение данных Yandex Forms, которые интегриуются с ALbato через специальный сгенерированный в Albato URL (web-hook).
```
